Provides support for viewing keystore details.
Core functionality was taken from keystore-explorer.


Important: Next version will have a price 3 USD Individual price or 5 USD Business price per year, to support my work
Features:
  • View keystore details
  • Supported keystore extensions: jks, jceks, bks, p12, uber, bcfks, pfx
  • Supported certificate extensions: pub, key, pem, cer, crt, p7, p7b, pkipath, spc, p10, spkac, pkcs8, pvk, crl
